COMANDOS QUE DEVEM SER DIGITADOS PARA RODAR O PYTHON NO AMBIENTE VIRUTUAL
>> através do CMD, acessar a pasta onde está o arquivo python que vai ser executado:
>> Digitar: py -3 -m venv .venv
>> Digitar: .venv\Scripts\activate
>> Digitar: python "Nome_do_Arquivo.py"

>> pip install flask flask-login flask-sqlalchemy
